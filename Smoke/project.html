OpenGL Smoke Animation

The aim of this project is to create Smoke Particles with the use of OpenGL graphics library. This would represent the individual project of COMP 371 in Concordia University.

As an Exchange Student, it has been the first time working with OpenGL and C++, as well as 3D. From the beginning I was aware that I would have to put all my efforts in this course. I consider that I have mainly learned and enjoyed the course, which I consider very important.
